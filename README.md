# Car Dealership Inventory System (LinkedList Version)

## Overview
This project is a Java-based application that simulates a car dealership inventory system. It allows users to manage vehicle records, including adding, removing, searching, and organizing car data using a custom linked list structure.

The goal of this project is to demonstrate core object-oriented programming concepts, data structures, and real-world data handling in a structured and scalable way.

## Features
- Add, remove, and update vehicle records
- Search inventory by different attributes (make, model, price, etc.)
- Store and manage data using a LinkedList structure
- Organized and structured data processing
- Input handling and validation

## Technologies Used
- Java
- Maven
- Object-Oriented Programming (OOP)
- Data Structures (Linked Lists)
- File Handling (if applicable)

## Project Structure
- `Car` → Represents each vehicle object
- `GenericLinkedList` → Custom linked list implementation
- `Main` → Handles user interaction and program flow

## Future Improvements
- Convert LinkedList to HashMap for faster lookup (O(1) access)
- Add file-based persistence for saving/loading data
- Implement sorting and filtering features
- Add a graphical user interface (GUI)
